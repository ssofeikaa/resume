## 📂 Проекты

### 1. Хедонические цены на итальянское вино
`econometrics_wine_pricing.pdf`

Исследование ценообразования на вине итальянского маркетплейса Tannico.

| | |
|---|---|
| **Данные** | 840 наблюдений, собраны парсингом с Tannico |
| **Методы** | OLS, 2SLS (инструментальные переменные) |
| **Задача** | Проверка 3 гипотез о факторах цены: регион, сортовой состав, рейтинг |

---

### 2. Влияние стриминг-подписки на продолжительность сна
`ml_streaming_sleep.pdf`

Оценка причинно-следственного эффекта стриминговых сервисов на качество сна с применением современных методов каузального вывода.

| | |
|---|---|
| **Данные** | Симулированные (кастомный DGP) |
| **Методы** | Классификация, регрессия, IPW, Doubly Robust, DML, IV |

---

### 3. Денежно-кредитная политика и макроэкономика России: репликация Uhlig (2005)
`time_series_monetary_policy.pdf`

Байесовский структурный VAR со знаковыми ограничениями на российских данных 2015–2025 — период инфляционного таргетирования.

| | |
|---|---|
| **Данные** | ВВП, ИПЦ, экспорт, резервы, госдолг, RUONIA · Росстат / ЦБ РФ |
| **Методы** | BSVAR, знаковые ограничения, IRF, FEVD · пакет VARsignR (R) |
| **Основа** | Uhlig H. (2005), *Journal of Monetary Economics* |
